## Description

Add support for tracking offline stores in global config (~/.bodega/config.yaml):

## Config structure
```yaml
offline_stores:
  git-a1b2c3d4e5f6: my-project
  path-9f8e7d6c5b4a: personal-tasks
```

## Functions to add to src/bodega/config.py
1. `get_offline_store_mapping() -> dict[str, str]`
   - Load ~/.bodega/config.yaml
   - Return offline_stores dict or empty dict

2. `set_offline_store_mapping(identifier: str, name: str) -> None`
   - Load ~/.bodega/config.yaml (create if doesn't exist)
   - Add/update offline_stores[identifier] = name
   - Write back to file

3. `list_offline_stores() -> list[tuple[str, str, Path]]`
   - Return list of (identifier, name, path) for all offline stores
   - Include existence check

## Usage
- Called during `bodega init --offline`
- Used by future management commands
